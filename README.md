# TackList 日程管理

一款可以自己部署的日程管理系统，帮你规划每一天、保持专注、把碎片时间用起来。通过浏览器即可使用，数据完全保存在你自己的电脑上，企业内网也可使用。

## 功能介绍

### 一、四种视图，随心切换

系统提供四种查看任务的方式，一键切换，数据实时同步：

- **日程视图**：按日期分组，以时间线形式展示每一天的任务，今天会高亮显示，一眼看清近期安排。
- **周视图**：7 列网格，每周一页，全天任务和具体时间的任务分别显示，支持拖拽调整日期。
- **月视图**：整月日历，每个格子里显示当天的任务摘要，超过 3 条会提示"还有 N 条"，方便总览全局。
- **四象限视图**：基于著名的艾森豪威尔矩阵，按"重要"和"紧急"两个维度把任务分到四个象限（重要且紧急、重要不紧急、紧急不重要、不重要不紧急），帮你一眼判断该先做什么。支持拖拽任务跨象限调整属性，还能拖动象限标题栏自定义排列顺序。

### 二、任务管理，井井有条

- **创建任务**：填写标题、选择清单分组、设置时间和提醒、标记重要/紧急、添加备注。界面采用渐进式设计，常用功能一目了然，高级设置按需展开。
- **灵活的时间设置**：支持"具体时间"和"时间段"两种模式。不填时间则自动视为全天任务，填了时间自动切换。提供"清除时间"按钮可随时恢复为无日期。
- **15 分钟步进时间选择器**：分钟选项固定为 00、15、30、45，首次展开自动定位到当前时间之后的下一个可选时刻，减少滚动查找。
- **清单分组**：自定义分类（如工作、生活、学习），每个清单可设置颜色标记。支持新增、编辑、删除，删除时分组内的任务会自动迁移到默认清单。
- **任务操作**：标记完成、编辑、删除都带防误操作机制——点击删除后按钮会变成红色的"确认删除"，3 秒后自动恢复。
- **拖拽操作**：在日程、周、月视图中拖拽任务可以修改日期；在四象限视图中拖拽可以修改重要/紧急属性。
- **顺延任务**：有过期未完成任务时，顶部会出现橙色"顺延任务"按钮，一键把所有过期任务延期到今天，时间点保持不变。
- **重复任务**：支持每天、每周、每月、每年或自定义间隔重复，省去重复创建的麻烦。

### 三、番茄专注，提高效率

内置番茄工作法计时器，帮你保持专注：

- **全屏专注页面**：左右双栏布局，左侧是大号倒计时圆环和开始/暂停按钮，右侧显示今日专注次数、专注时长、总专注次数以及最近 20 条专注历史。
- **任务关联**：开始专注前可以选择一个任务进行关联，专注完成后自动记录到该任务的历史中。
- **自动循环**：可开启"自动休息"和"自动专注"，专注结束自动进入休息，休息结束自动进入下一轮专注。
- **侧边栏倒计时**：计时运行时，侧边栏底部会显示实时倒计时和当前任务名称，点击即可跳转到专注页面。
- **完成提醒**：专注完成和休息结束时都会播放提示音并发送系统通知。

### 四、智能提醒，不再遗漏

- **系统级通知**：任务快开始时，系统会弹出桌面通知，即使浏览器最小化也能看到。
- **可配置提前时间**：在设置中可选择提前 5 分钟、1 天或自定义分钟数提醒，设为 0 则关闭提醒。
- **声音提醒**：通知出现时同时播放提示音。
- **每个任务只提醒一次**：避免重复打扰。
- **通知权限管理**：在设置中可以查看当前通知权限状态，并一键请求授权。

### 五、正念小事，关注自己的成长

不知道做什么的时候，点一下"正念小事"按钮：

- 系统会从内置 220+ 条小事库中随机推荐一条，都是10分钟内能完成、锻炼自己的正念思维的小事。
- 每次会话最多可查看 4 条小事，可以逐条"换一个"，也可以在最后从已展示的 4 条里挑选一条。
- 点击"接受，去做！"后，该小事会自动变成今天的任务加入清单。
- 涵盖办公场景和日常生活场景两类内容。

### 六、任务清单视图与计划面板

- **任务清单视图**：以分组列表形式展示所有任务，自动按"已过期、今天、明天、最近 7 天、更远、无日期、已完成"分类排序，过期分组会用红色醒目标注。
- **计划面板**：从右侧滑入的面板，按分组显示所有未完成任务，方便快速浏览和拖动任务规划。

### 七、设置中心，个性化定制

提供丰富的设置选项，让系统贴合你的习惯：

- **外观**：浅色/深色主题一键切换，支持上传自定义背景图片并调整透明度。
- **新建任务默认值**：可设置默认清单、默认重要/紧急、默认持续时长，减少重复操作。
- **视图偏好**：默认首页视图（日程/周/月/四象限）、周起始日（周一或周日）、是否显示已完成任务。
- **番茄计时**：专注时长（1-60 分钟）、休息时长（1-30 分钟）、自动休息、自动专注。
- **通知设置**：消息显示时长、任务提前提醒分钟数、通知权限状态。
- **无聊小事**：是否在接受后自动创建任务。
- **数据管理**：导出数据、导入数据、自动备份开关、备份周期、数据保留周期。

### 八、节日与农历

内置中国法定节假日和调休日数据，并显示农历日期，方便安排日程。联网在线版还提供节假日数据通过 API 自动更新。

### 九、数据安全与备份

- **完全本地存储**：所有数据保存在你自己的电脑上，不经过任何第三方服务器。
- **导入导出**：随时将数据导出为 JSON 文件备份，也可导入备份恢复或迁移到其他设备。
- **自动备份**：可开启自动备份功能，按设定的周期（1-365 天）提醒你下载备份文件。
- **兼容旧版本**：导入旧版本数据时，新增的设置字段会自动填充默认值，不会丢失数据。

## 安装与使用

### 环境要求

- **Python 3.8 或更高版本**（无需安装任何额外依赖包）
- **现代浏览器**：Chrome、Firefox、Edge、Safari 均可

### Windows 安装

1. 将项目文件夹复制到目标位置（路径中请避免中文或空格），例如 `C:\Users\你的用户名\TackList`
2. 双击 `install.bat`，脚本会自动检查 Python 环境、创建桌面快捷方式和开机自启动项
3. 安装完成后，双击桌面上的"日程管理"快捷方式即可启动，浏览器会自动打开

**手动启动**（如不运行安装脚本）：

```cmd
cd C:\Users\你的用户名\TackList
start.bat
```

**常用命令**：

```cmd
start.bat           # 启动服务
start.bat stop      # 停止服务
start.bat restart   # 重启服务
```

### Linux 安装

1. 将项目文件夹复制到用户目录：

```bash
cp -r /path/to/TackList ~/TackList
cd ~/TackList
chmod +x tacklist.sh install.sh autostart.sh
```

2. 运行安装脚本（自动创建桌面快捷方式和开机自启动项）：

```bash
./install.sh
```

3. 或手动启动：

```bash
./tacklist.sh start
```

**常用命令**：

```bash
./tacklist.sh start     # 启动服务
./tacklist.sh stop      # 停止服务
./tacklist.sh restart   # 重启服务
./tacklist.sh open      # 启动服务并打开浏览器
```

### 快速启动（任意平台）

只要装了 Python，都可以直接运行：

```bash
python3 server.py
```

然后在浏览器打开 `http://localhost:14438` 即可。

### 访问地址

启动后浏览器会自动打开。如需手动访问，默认地址是 `http://localhost:14438`。若端口被占用，系统会自动寻找下一个可用端口，可在 `server.log` 中查看实际使用的端口。

## 离线使用

系统支持完全离线运行：

- 项目自带 `index_offline.html` 离线版本，使用本地的样式和字体资源，无需联网。
- 所有数据保存在本地的 `data.json` 文件中。
- 不需要安装任何第三方 Python 包。

## 卸载

### Windows

1. 执行 `start.bat stop` 停止服务
2. 删除桌面上的"日程管理"快捷方式
3. 按 `Win+R`，输入 `shell:startup`，删除自启动快捷方式
4. 删除项目文件夹（可选，会删除所有数据）

### Linux

```bash
./tacklist.sh stop
rm -f ~/桌面/日程管理.desktop
rm -f ~/.config/autostart/日程管理.desktop
rm -rf ~/TackList
```

## 常见问题

**Q：启动后浏览器没有自动打开？**
手动在浏览器地址栏输入 `http://localhost:14438` 即可。

**Q：提示"未找到 Python"？**
请确认安装 Python 时勾选了"Add Python to PATH"，或手动将 Python 安装路径添加到系统环境变量中。

**Q：端口被占用？**
系统默认使用端口 14438，可在设置中修改。若端口被占用，系统会自动递增尝试下一个端口。查看 `server.log` 获取实际端口：
- Windows：`type server.log | findstr "TackList Server"`
- Linux：`cat server.log | grep "TackList Server"`

**Q：如何取消开机自启动？**
- Windows：按 `Win+R`，输入 `shell:startup`，删除"日程管理"快捷方式。
- Linux：删除 `~/.config/autostart/日程管理.desktop` 文件。
